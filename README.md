# Sudoku
Gra napisana w c# przy użyciu frameworka Windows Forms.
## Utrudnienia
### Tryby trudności
- Łatwy
- Średni
- Trudny
### Licznik błędów
Gra liczy każdy popełniony błąd.
Max 3 błędy.
## Dodatkowe funkcjonalności
- Licznik czasu
- Licznik rund
- Najlepsze czasy
